AnchorCodified Context: Infrastructure for AI Agents in a Complex Codebase
arXiv:2602.20478 · Vasilopoulos (Independent), 2026 · systems paper + experience report · cite: Vasilopoulos '26
Defines codified context infrastructure as a three-tier scheme — a hot-memory "constitution," domain-specialist sub-agents routed by trigger tables, and a cold-memory knowledge base served over MCP — built solo across 283 Claude Code sessions on a 108k-line C# codebase. The spine of the thread; the only paper whose references were pre-harvested.
